I am designing a solution to deploy a static web app using Nginx on Google Cloud Platform. To automate the deployment process, I will be using Continuous Integration and Continuous Deployment (CI/CD) principles with Google Cloud Build and Google Cloud Deploy. This project will involve creating a Docker image for the Nginx web server, pushing it to Google Container Registry, and then deploying it to Google Cloud Run for hosting the static web app. By: Lalit Kumar Gautam
